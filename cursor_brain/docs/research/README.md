# Research Templates

The original repository contained private research libraries, round notes, and family-level summaries. Those materials were removed from the public release and replaced by generic templates.

Use this directory to store sanitized assets such as:

- public-safe experience summaries
- template lesson libraries
- branch-level research notes without real ids or expressions
- documentation about how research memory is structured

The file `public_experience_library_template.json` is included as a format example.
